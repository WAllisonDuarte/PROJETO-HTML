Topologia de Rede

A topologia de rede define a estrutura e a forma como os dispositivos estão interconectados. Os principais tipos são:

Barramento: Todos os dispositivos compartilham um único meio de transmissão.

Anel: Os dispositivos estão conectados em um círculo fechado.

Estrela: Todos os dispositivos se conectam a um ponto central.

Malha: Cada dispositivo está conectado a vários outros.

Híbrida: Combina diferentes topologias.

Protocolos de Rede

Protocolos de rede são regras que definem a comunicação entre dispositivos. Alguns exemplos:

HTTP/HTTPS: Para comunicação web.

TCP/IP: Base da internet.

UDP: Protocolo rápido para transmissão de dados.

DNS: Resolve nomes de domínios para endereços IP.

FTP/SFTP: Transferência de arquivos.

Endereçamento de Rede

Define como os dispositivos são identificados na rede.

IPv4: Formato padrão com endereços de 32 bits.

IPv6: Substitui o IPv4, usando 128 bits.

MAC Address: Endereço físico da interface de rede.

Dispositivos de Rede

Os dispositivos conectam e gerenciam o tráfego de rede:

Roteador: Encaminha pacotes entre redes.

Switch: Gerencia conexões locais.

Modem: Converte sinais digitais e analógicos.

Access Point: Expande a rede sem fio.

Tipos de Redes

Classificação com base na área de cobertura:

LAN (Local Area Network): Pequena área.

MAN (Metropolitan Area Network): Abrange uma cidade.

WAN (Wide Area Network): Redes globais, como a internet.

PAN (Personal Area Network): Redes pessoais, como Bluetooth.

Segurança de Redes

Principais conceitos:

Firewall: Filtra tráfego de rede.

Criptografia: Protege dados durante a transmissão.

VPN: Garante conexões seguras.

Autenticação: Controle de acesso.

Comutação de Pacotes vs. Comutação de Circuitos

Comutação de Pacotes: Dados divididos em pacotes que são transmitidos independentemente.

Comutação de Circuitos: Requer conexão dedicada antes da transmissão.

Largura de Banda e Latência

Largura de Banda: Quantidade de dados que podem ser transmitidos em um tempo.

Latência: Tempo de resposta entre envio e recebimento de dados.

Tecnologias de Rede

Principais tecnologias utilizadas:

Wi-Fi: Conexão sem fio.

Ethernet: Rede com fio.

5G: Alta velocidade para dispositivos móveis.

Fibra Óptica: Conexão de alta velocidade por meio de pulsos de luz.
